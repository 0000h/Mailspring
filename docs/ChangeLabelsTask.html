Summary

Create a new task to apply labels to a message or thread.

Takes an options object of the form:

  • labelsToAdd: An Array of Labels or Label ids to add
  • labelsToRemove: An Array of Labels or Label ids to remove
  • threads: An Array of Threads or Thread ids
  • messages: An Array of Messages or Message ids
