Summary

MultiselectList wraps {ListTabular} and makes it easy to present a {ModelView} with selection support. It adds a checkbox column to the columns you provide, and also handles:

  • Command-clicking individual items
  • Shift-clicking to select a range
  • Using the keyboard to select a range
